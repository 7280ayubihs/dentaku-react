# 電卓（ReactJS）
React学習として作成した電卓です。<br>
次のURLから利用ください。<br>
https://7280ayubihs.github.io/dentaku-react/

## 起動方法（ローカル実行）
1. `npm install` でパッケージをインストール
2. `npm start` で開発サーバ起動
3. `http://localhost:3000/` にアクセスして電卓を利用する。
